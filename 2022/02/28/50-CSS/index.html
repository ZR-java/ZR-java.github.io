## 2.内部（嵌入）样式表

**使用一个选择器将样式定义在head标签体中，并使用 style包裹**

```html
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style>
        div{
            background-color: black;
            color:orange;
            font-family: 宋体;
        }
    </style>
</head>
<body>
    <div>
        内部(嵌入)样式!!!
    </div>
</body>
</html>

    3.外部（引入）样式表

    首先编写一个style.css文件

    div{
	background-color: black;
	color:orange;
	font-family: 宋体;
}

    编写一个使用到这个样式表文件的网页文件，假设该网页文件与style.css在同一个目录下，则内容如下

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <link type="text/css" rel="stylesheet" href="style.css"/>
</head>
<body>
    <div>外部(引入)样式</div>
</body>
</html>

    外部样式表会提高网页的浏览速度

    • 浏览器会缓存样式文件
    • 每次加载新的网页时，如果该网页仅仅只有要显示的内容， 而没有显示的样式，这样势必会比内联、嵌套样式表的方式要快
    • 推荐使用外部（引入）样式表的方式

    三种样式的优先级

    三种方式的设置样式，总样式应该是三种样式之和

    相同样式的优先级：

    • 内联样式优先级最高
    • 内部样式和外部样式，谁在文件中定义的位置靠后，就会覆盖前面的定义，从而起作用

    三、样式规则的选择器

    1.html选择器

    • 以html元素本身作为的选择器，如p（对应html的<p></p>段落标签），input（对应html的<input></input>输入标签）
    • 当前网页中的所有的该元素标签的显示效果，都按照html 选择器中所定义的方式来显示
    <html>
    <head>
        <meta charset="UTF-8">
        <style>
            p{
                color:blue;
                font-size:30px;
            }
        </style>
    </head>
    <body>
        <p>
            这是一个段落!
        </p>
    </body>
</html>

    2.class选择器

    • 在同一个网页中可以有多个元素标签的class属性会取相同的值，如此同名的元素标签将一起受class选择器里面样式的定义影响
    • 类别选择器的定义需要以点号（.）开头
    <html>
    <head>
        <style>
        	.cls1{
                color: red;
            }
            div{
                color: blue;
            }
        </style>
    </head>
    <body>
        <h3 class="cls1">选择器</h3>
        <div>	<!--这个div显示蓝色-->
            div1
        </div>
   
        <span class="cls1">中国</span>
        <br>
        <div class="cls1">	<!--这个div显示红色-->
            <span>中华人民共和国</span>
        </div>
    </body>
</html>

    3.id选择器

    • id属性用来定义某一个特定的html元素标签
    • 一个网页中元素标签的id属性是唯一的
    • id选择器的定义需要以井号（#）开头
    <html>
    <head>
        <style>
            #green{
                color:green;
            }
        </style>
    </head>
    <body>
        <div>
            <span id="green">中华人民共和国</span>
        </div>
    </body>
</html>

    4.关联选择器(父子关系)

    • 用一个空格的两个或者更多的单一选择器组成的字符串

    • 例如

      • p em{background：yellow}
      • 其中的”p em”就是关联选择器，它表示段落中的强调文本（<em></em>标签对中的内容）的背景为黄色，而其他地方出现的强调文本不受影响

    • 关联选择器定义的样式规则的优先级要大于单一选择器定义的样式规则

      • p em{background：yellow}
      • em{background：blue}
      • 段落中的强调文本的背景色为黄色

    5.组合选择器(平级关系)

    • 为了减少样式表的重复声明，可以在一条样式规则定义语 句中组合若干个选择器，每个选择器之间用逗号隔开
    • 例如：h1,h2,h3,h4,h5,hr{color: red}

    6.伪元素选择器

    • 对同一个html元素的各种状态和其所包括的部分内容的一 种定义形式
    • 例如：对于超链接（<a></a>）的正常状态（没有任何动作之前）、访问过的状态、选中状态、光标移动到超链接文本上的状态，对于段落的首字母和首行， 都可以使用伪元素选择器来定义

    伪元素作为选择器的样式规则的定义格式

    • html元素：伪元素（属性：属性值）

    常用的伪元素

    • a:link（超链接的正常状态）
    • a:hover（光标移动到超链接上时的状态）
    • a:active（选中超链接时的状态）
    • a:visited（访问过的超链接的状态）
    • p:first-line（段落中的第一行文本）
    • p:first-letter（段落中的第一个字母）

    类选择器和伪元素选择器一起使用

    • html元素.类名: 伪元素{属性: 属性值}

    伪元素选择器

    • a:link,a:visited{text-decoration:none} 超链接无下划线
    • a.class1:link,a.class1:visited{text-decoration:underline}

    四、样式的常用属性

    1.颜色

    color

    • 定义文本的颜色
    • 取值
      • color:green
      • color:#ff6600，color:#f60（简写形式）
      • color:rgb(0,255,0)
      • color:rgba(182,22,206,0.8)
    • RGB
      • 红(R)、绿(G)、蓝(B) 每个的取值范围0~255
    • RGBA
      • RGBA是代表Red（红色）Green（绿色）Blue（蓝色）和Alpha的（色彩空间）透明度

    2.字体

    font-size

    • 字体大小
    • 取值
      • px：具体的像素值
      • %（百分比）：相对于父标签字体的大小，如果父标签的字体大小设置为40px，则 子标签字体大小的值为50%，则实际上是20px；如果子标签字体的大小的值为200%，则实际上是80px，它是一个相对的值
      • smaller：子标签字体比父标签字体更小
      • larger：子标签字体比父标签字体更大
      • inherit：子标签字体继承父标签字体大小，也就是父子标签的字体大小一样

    font-family

    • 定义字体
    • 取值
      • “”微软雅黑”，”宋体””
      • 可以使用“,”隔开，以确保当字体不存在的时候直接使用下一个

    font-weight

    • 字体加粗
    • 取值
      • normal（默认值）、bold（粗）、bolder（更粗）、lighter（更细）
      • 100、200、300、400、500、600、700、800、900
        • 400 = normal，700 = bold
        • 只能是上述的9个整数的百值，不能使用其他数字

    font-style

    • 定义字体的样式
    • 取值
      • normal（标准）、italic（斜体）、oblique（倾斜）、inherit（继承）

    font-variant

    • 定义字母大小写的显示样式
    • 取值
      • normal：标准
      • small-caps：小型大写字母显示格式
      • inherit：继承

    3.背景

    background-color

    • 定义背景颜色
    • 取值
      • color:green
      • color:#ff6600，color:#f60（简写形式）
      • color:rgb(0,255,0)
      • color:rgba(182,22,206,0.8)

    background-image

    • 定义背景图片
    • 取值
      • url（图片路径）

    background-repeat

    • 定义背景图片的重复方式
    • 取值
      • repeat：重复平铺满，默认
      • repeat-x：向x轴重复
      • repeat-y：向y轴重复
      • no-repeat：不重复

    background-position

    • 定义背景图片的位置
    • 取值
      • 横向：left、center、right
      • 纵向：top、center、bottom
      • 也可以使用像素值，表示距离左边、右边有多远
    • 横向和纵向要一起取值，如果只取一个值，则会在该值的后面默认加上一个center

    background

    • 定义背景相关样式
    • 上面的格式background-xxx只能为某一种样式定义，如果全部都要定义比较繁琐，可以使用background样式一起定义
    • 以上所有背景的样式可以一起简写
    • 格式
      • 背景颜色 url （图像）重复 位置
      • background：#f60 url(images/bg.jpg) no-repeat top center

    4.文本

    text-align

    • 定义文本的横向排列方式
    • 取值
      • left、center、right

    line-height

    • 定义文本的高度
    • 取值
      • %（百分比）：相对于字体大小的百分比行高
      • px
        • 像素值所定义的行高
        • 默认值为0

    text-indent

    • 定义文本首行缩进的方式
    • 取值
      • %（百分比）：相对于父容器宽度的百分比进行缩进
      • px
        • 像素值是多少，就缩进多少个像素
        • 默认值为0
      • inherit：继承

    letter-spacing

    • 定义文字之间的距离
    • 取值
      • normal：默认
      • px：设置具体的像素数值（可以设置负值，会出现重叠的现象）
      • inherit：继承

    word-spacing

    • 定义单词之间的距离
    • 取值
      • normal：默认，标准距离
      • px：像素值
      • inherit：继承

    direction

    • 定义文本的方向
    • 取值
      • ltr
        • left to right 的缩写，从左到右
        • 默认
      • rtl
        • right to left 的缩写，从右到左
      • inherit：继承

    text-transform

    • 定义文本的大小写
    • 取值
      • none：默认
      • capitalize：每个单词首字母大写
      • uppercase：所有字符全部大写
      • lowercase：所有字符全部小写
      • inherit：继承，根据父元素的text-transform值来显示

    5.边框

    border-style

    • 定义边框四周的风格
    • 取值
      • none：无边框
      • solid：直线边框
      • dashed：虚线边框
      • dotted：点状边框
      • double：双线边框
      • groove：groove边框
      • ridge：ridge边框
      • inset：inset边框
      • outset：outset边框
      • inherit：继承
    • 如果需要对边框的上、下、左、右分别定义，可以使用如下的样式
      • border-top-style：上边边框样式
      • border-bottom-style：下边边框样式
      • border-left-style：左边边框样式
      • border-right-style：右边边框样式

    border-width

    • 定义边框的宽度
    • 取值
      • thin：细边框
      • medium：中等边框
      • thick：粗边框
      • px：固定像素值的边框
      • inherit：继承
    • 如果需要对边框的上、下、左、右分别定义宽度，可以使用如下样式
      • border-top-width：上边边框宽度
      • border-bottom-width：下边边框宽度
      • border-left-width：左边边框宽度
      • border-right-width：右边边框宽度

    border-color

    • 定义边框的颜色
    • 取值
      • color:green
      • color:#ff6600，color:#f60（简写形式）
      • color:rgb(0,255,0)
      • color:rgba(182,22,206,0.8)
    • 如果需要对边框的上、下、左、右分别定义颜色，可以使用如下的样式
      • border-top-color：上边边框颜色
      • border-bottom-color：下边边框颜色
      • border-left-color：左边边框颜色
      • border-right-color：右边边框颜色
    • 属性值的四种情况
      • 一个值
        • border-color：（上、下、左、右）
      • 两个值
        • border-color：（上下）（左右）
      • 三个值
        • border-color：（上）（左、右）（下）
      • 四个值
        • border-color：（上）（右）（下）（左）

    简写方式

    • border：样式 宽度 颜色
    • border：solid 2px blue

    6.列表

    list-style-type

    • 定义列表项标记的样式
    • 有序列表和无序列表都可以使用的样式
    • 取值
      • none：没有字符图案，无序列表默认的实心圆点和有序列表默认的数字都没有了
      • disc：默认，无序列表默认是实心圆点，有序列表默认是数字
      • circle：空心圆
      • square：实心方块
      • decimal：数字
      • decimal-leading-zero：0开头的数字（01、02、03… )
      • lower-roman：小写罗马数字（i、ii、iii、iv、v…）
      • upper-roman：大写罗马数字（I、II、III、IV、V…)
      • lower-alpha：小写英文字母（a、b、c、d、e…）
      • upper-alpha：大写英文字母（A、B、C、D、E…）
      • lower-greek：小写希腊字母（alpha、beta、gamma…）
      • lower-latin：小写拉丁字母（a、b、c、d、e…）
      • upper-latin：大写拉丁字母（A、B、C、D、E…）
      • hebrew：传统的希伯来编号
      • armenian：传统的亚美尼亚编号
      • georgian：传统的乔治亚编号（an、ban、gan…）
      • cjk-ideographic：简单的表意数字
      • hiragana：小写的日本语元音编号（a、i、u、e、o、ka、ki…）
      • katakana：大写的日本语元音编号（A、I、U、E、O、KA、KI…）
      • hiragana-iroha：小写的日本语元音编号（i、ro、ha、ni、ho、he、to…）
      • katakana-iroha：大写的日本语元音编号（I、RO、HA、NI、HO、HE、TO…）

    list-style-position

    • 定义列表项标记的位置
      • inside：列表项目标记放置在文本以内，且环绕文本根据标记对齐
      • outside：默认值，保持标记位于文本的左侧。列表项目标记放置在文本以外， 且环绕文本不根据标记对齐
      • inherit：继承

    list-style-image

    • 定义列表项标记的图片
      • URL：图像的路径
      • none：默认，无图像被显示
      • inherit：继承

    简写方式

    • list-style-：样式 位置 图片
    • list-style:square inside url(‘images/bg.jpg’)

    五、DIV+CSS布局

    1.div和span

    • div和span在整个html标签中没有任何意义，他们的存在就是为了应用css样式
    • div和span的区别在于span是内联元素，div是块级元素

    2.盒子模型

    • margin：盒子外边距
    • padding：盒子内边距
    • border：盒子边框宽度
    • width：盒子宽度
    • height：盒子高度

    3.布局相关属性

    z-index

    • 定义层的优先级顺序
    • 默认值为0，数值越大越在上面

    display

    • 定义层的显示方式
    • 取值
      • none：不显示
      • block：
        • 块状显示，在元素的后面换行显示下一个元素
        • 可以实现span到div的显示效果的转换
        • 此时设置元素的宽与高属性能起效果
      • inline：
        • 内联显示，在元素的后面空格显示下一个元素，多个这种样式的元素可以实现一行显示
        • 可以实现div到span显示效果的转换
        • 此时再设置元素的宽与高属性已经无效

    float

    • 定义层的浮动
    • 取值
      • left：向左浮动
      • right：向右浮动

    clear

    • 定义层清除浮动
    • 取值
      • both：清除全部的浮动

    overflow

    • 定义层的溢出，如果层里面的内容超出了层的大小，可以指定层内部内容的显示方式
    • 取值
      • hidden：隐藏超出层大小的内容
      • scroll：无论层内部的内容有没有超出层的大小，始终会显示一个滚动条
      • auto：当层内部的内容超出层的大小才会显示一个滚动条
