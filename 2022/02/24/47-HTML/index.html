    HTML简介

    html

    • HTML是英文Hyper Text Markup Language（超文本标记语言）的缩写

    网页

    • 显示使用HTML编写好文件，该文件一般以html、htm、 xhtml为后缀名

    不同的浏览器之间的不兼容问题

    • Firefox
    • IE
    • Chrome
    • Opera

    一个同样的标签，可能实现额外的功能或者显示

    由谁来制定规则

    IETF W3C

    当前版本

    HTML4.01 XHTML HTML5

    html的保存格式

    .html .htm .xhtml

    HTML的标签和属性

    标签和标签内部的内容构成html文档

    • 成对标签
      • 以<标签名>开始，以</标签名>结束
      • 又称之为容器标签，一对标签的内部还可以嵌套其他的标签，这些所有内部的标签统称为该标签的标签体
    • 单标签
      • 无标签体的标签
    • html元素
      • 一个html标签以及该标签的标签体的总称

    html的属性

    • 标签的属性，就是用来控制我们的内容（图像、文本等的） 如何的显示

    • 格式

      • 属性名=属性值
      • 属性值可以使用单引号、双引号包裹起来

    • 例如

      • <body bgcolor="red">内容</body>
      • 以上的属性是用来控制我们的网页的背景颜色，bgcolor就是body的属性，他的值是red（红色）

    标签、属性、属性值建议使用小写

    标签对大小写不敏感（不区分大小写）

    <HTML> <Html> <html>都是定义相同的标签

    HTML的注释

    注释一段内容时使用 <!--注释内容-->

    不支持嵌套

    HTML的代码格式

    空格键和回车键在网页中都不会起到任何作用，我们为了让代码清晰易读，可以使用空格和回车键进行编排

    注意：缩进时保持严格的规则，以”Tab”键进行缩进

    HTML的实体

    HTML的基本结构

    html标签

    • 含义
      • html文档的文档标签，也称为html开始标签
    • 功能
      • 这对标签分别位于网页的最前端和最后端
      • 在最前端表示网页的开始
      • 在最后端表示网页的结束

    head标签

    • 含义
      • head文件头标签，也称为head头信息开始标签
    • 功能
      • 用来包含文件的基本信息，比如网页的标题、关键字，在<head></head>内可以放<title></title>、<meta></meta> 、<style></style>等标签
    • 注意
      • 在<head></head>标签内的内容不会在浏览器中显示

    title标签

    • 含义
      • html文件标题标签
    • 功能
      • 网页的”主题” ，显示在浏览器的窗口的左上边
    • 注意
      • 网页的标题不能太长，要短小精悍，能具体反应页面的内容，<title></title>标记中不能包含其他标记

    meta标签

    • 含义
      • 页面的元信息（meta-information）
    • 功能
      • 提供有关页面的元信息（meta-information），比如针对搜索引擎和更新频度的描述和关键词
    • 必须的属性
      • content，值为name属性的相关描述
    • 常见的属性
      • author、keyword、description、others
      • 例如
        • 比如描绘出网页的关键词：<meta name="author" content="ZR">
    • 注意
      • meta标签必须放在head元素里面

    body标签

    • 含义
      • html文档的主体标签
    • 功能
      • <body>...</body>是网页的主体部分，在此标签之间可以包含如<p></p>、<h1></h1>、<br>、<hr>等标签，正式由这些内容组成了我们看见的网页
    • 常见属性
      • bgcolor：设置背景颜色
        • <body bgcolor="red"></body>
      • text：设置文本颜色
        • <body text="green"></body>
      • link：设置链接的颜色
        • <body link="blue"></body>
      • vlink：设置已经访问过了的链接的颜色
        • <body vlink="yellow"></body>
      • alink：设置正在被点击的链接的颜色
        • <body alink="red"></body>

    HTML的文档标签

    html的格式标签

    br

    • 强制换行标签
    • 让后面的文字、图片、表格等等，显示在下一行

    p

    • 段落标签
    • 段落，由于多个空格和回车在HTML中会被等效为一个空格，所以HTML中要换段落就要用<p>，<p>段落中也可以包含<p>段落
    • align属性
      • left center right

    center

    • 居中对齐标签
    • 让段落或者是文字相对于父标签居中显示

    pre

    • 预格式化标签
    • 保留预先编排好的格式

    li

    • 列表项目标签
    • 每一个列表使用一个
    • 标签

    ul

    • 无序列表标签
    • 声明这个列表没有序号

    ol

    • 有序列表标签

    • 可以显示特定的一些顺序

      • 格式

        • <ol type="符号类型">
    <li type="符号类型"></li>
    <li type="符号类型"></li>
</ol>

      • 有序列表的type属性值

        • 1：阿拉伯数字1,2,3等，默认的type属性值
        • A：大写字母A，B，C等
        • a：小写字母a，b，c等
        • I：大写罗马数字Ⅰ，Ⅱ，Ⅲ，Ⅳ，Ⅴ等
        • i：小写罗马数字i，ii，iii，iv，v等

      • value：指定一个新的序列数字起始值

      • 列表可以进行嵌套

    dl、dt、dd

    • <dl></dl>:创建普通的列表
    • <dt></dt>:创建列表的上层项目
    • <dd></dd>:创建列表的下层项目

    hr

    • 水平分割线标签
    • 段落之间的分割线

    div

    • 分区显示标签，也称之为块标签
    • 常用来编排一大段的HTML段落，也可以用于格式化表， 和

      很相似

    • 块可以多层嵌套使用

    span

    • 分区显示标签
    • 和div相似，但是没有换行作用.

    html的文本标签

    hn

    • 标题标签
    • 共有6个级别，n的范围1~6，不同级别对应显示大小不同的标题

    font

    • 字体设置标签
    • 设置字体的格式
    • 常用属性
      • size（字体大小）1~7px
      • color（颜色）
      • face（字体）

    • b 粗字体标签

    • i 斜字体标签

    • sub 文字下标字体标签

    • sup 文字上标字体标签

    • tt 打印机字体标签

    • cite 引用方式的字体，通常是斜体

    • em 表示强调，通常显示为斜体字

    • strong 表示强调，通常显示为粗体字

    • small 小型字体标签

    • big 大型字体标签

    • u 下划线字体标签

    HTML的图像标签

    img标签

    • 格式
      • <img src="路径/文件名.图片格式" width="属性值" height="属性值" border="属性值" alt="属性值">
    • 常用属性
      • src：指定我们要加载的图片的路径和图片的名称以及图片格式
      • width：指定图片的宽度，单位px、em、cm、mm
      • height：指定图片的高度，单位px、em、cm、mm
      • border：指定图标的边框宽度，单位px、em、cm、mm
      • alt：
        • 当网页上的图片被加载完成后，鼠标移动到上面去，会显示这个图片指定的属性文字
        • 如果图像没有下载或者加载失败，会用文字来代替图像显示
        • 搜索引擎可以通过这个属性的文字来抓取图片
    • 注意
      • 为单标签，不需要使用闭合
      • 在加载图像文件的时候，文件的路径或者文件名文件格式错误，将无法加载图片

    HTML的超链接

    href属性

    • 格式
      • <a href="" target="打开方式" name="页面锚点名称">链接文字或者图片</a>
    • 常用属性
      • href
        • 链接的地址，链接的地址可以是一个网页，也可以是一个视频、 图片、音乐等等
      • target
        • 定义超链接的打开方式
        • 属性值
          • _blank：在一个新的窗口中打开链接
          • _self：在当前窗口中打开链接，默认值
          • _parent：在父窗口中打开页面（框架中使用较多）
          • _top：在顶层窗口中打开文件（框架中使用较多）
        • 注意：如果是在同一个frameset中，_parent与_top的效果是一 样的，只有在多个html文件中包含frameset，_parent与_top才 会有区别
      • name
        • 指定页面的锚点名称
    • 超链接的定位
      • 多个网页的定位
        • 取标记
          • <a name="mark">
          • <a href="6.html#mark">6.html</a>
      • 同一个网页内部的定位
        • <a href="#">自己</a>打开网页本身
        • <a href="">空白</a>打开网页本身

    HTML的表格

    table标签

    • 格式
      • <table 属性1="属性值1" 属性2="属性值2"... ...>表格内容</table>
    • 常用属性
      • width
        • 表示表格的宽度，它的值可以是像素（px）也可以是父级元素的百分比（%）
      • height
        • 表示表格的高度，它的值可以是像素（px）也可以是父级元素的百分比（%）
      • border
        • 表示表格外边框的宽度
      • align
        • 表格的显示位置
        • 取值：
          • left：居左显示
          • center：居中显示
          • right：居右显示
      • cellspacing
        • 单元格之间的间距，默认是2px，单位是像素
      • cellpadding
        • 单元格内容与单元格边框的显示距离，单位是像素
      • frame
        • 控制表格外边框最外层的四条线框
        • 取值：
          • void：无边框，默认值
          • above：仅有顶部边框
          • below：仅有底部边框
          • hsides：仅有顶部边框和底部边框
          • lhs：仅有左侧边框
          • rhs：仅有右侧边框
          • vsides：仅有左右侧边框
          • box：包含全部4个边框
          • border：包含全部4个边框
      • rules
        • 控制是否显示以及如何显示单元格之间的分割线
        • 取值：
          • none：无分割线，默认值
          • all：包括所有分割线
          • rows：仅有行分割线
          • clos：仅有列分割线
          • group：仅在行组和列组之间有分割线

    caption标签

    • 作用：给表格一个标题
    • 格式
      • <caption>属性的插入位置，直接位于<table>属性之后，<tr>表格行之前
    • 常用属性
      • align
        • 取值：
          • top：标题放在表格的上部
          • bottom：标题放在表格的下部
          • left：标题放在表格的左部
          • right：标题放在表格的右部

    tr标签

    • 作用：定义表格的一行，对于每一个表格行，都是由一对<tr>...</tr>标签表示，每一行<tr>标签内可以嵌套多个<td>或者<th>标签
    • 可选属性
      • bgcolor
        • 设置背景颜色
        • bgcolor=”颜色值”
      • align
        • 设置水平方向对齐方式
        • 取值
          • bottom：靠底部对齐
          • top：靠顶端对齐
          • middle：居中对齐
      • valign
        • 设置垂直方向对齐方式
        • 取值
          • left：靠左对齐
          • right：靠右对齐
          • center：居中对齐

    td和th标签

    • 作用：<td>和<th>都是单元格的标签，其必须嵌套在<tr>标签内，是成对出现的
    • 区别
      • <th>是表头标签，通常位于首行或者首列，<th>中的文字默认会被加粗，而<td>不会
      • <td>是数据标签，表示该单元格的具体数据
      • 两者的标签属性都是一样的
    • 常用属性
      • bgcolor
        • 设置单元格背景
      • align
        • 设置单元格水平对齐方式
      • valign
        • 设置单元格垂直对齐方式
      • width
        • 设置单元格宽度
      • height
        • 设置单元格高度
      • rowspan
        • 设置单元格所占行数
      • colspan
        • 设置单元格所占列数
