    一、什么是脚本程序和JavaScript

    什么是脚本程序

    • 相对于可编译的程序而言
    • 解释器

    什么是JavaScript

    • 前身叫做LiveScript，是一种基于对象和事件驱动、具有安全性能的脚本语言
    • JavaScript与Java是两个公司的产品，作用与用途大不一样，仅仅是语法相似
    • JavaScript、JScript、ECMAScript

    JavaScript的执行机制

    是逐行执行的!

    二、脚本代码的位置

    • 放置在<script></script>标签之间
    • script标签对可以出现在html标签的head标签中，执行的时机是显示网页之前.
    • script标签可以出现在html标签的body标签中，执行的时机是显示网页之时，如果在body标签中出现多个script标签对，则多个标签对之间可以互相访问定义的变量、函数.
    • script标签内部的JavaScript代码可以单独存放在后缀名为js的文件中， 再在html网页中引用该文件，这样会使得网页看起来更加的简洁
    • script标签内部的JavaScript代码可以出现标签属性值中
    • script标签内部的JavaScript代码，在早期，或者是某些浏览器还不认识，会误以为是html代码的一部分，会直接显示出来，为了避免这一现象的发生，可以使用html的注释（<!-- -->）将JavaScript代码注释起来

    代码详见Web/JavaScript/JS脚本的位置.html

    三、JavaScript的基本语法

    1.标识符

    • JavaScript中定义的符号，例如：变量名、函数名、数组名等。标识符可以由任意的大小写字母、数字、下划线和$组成，但是不能以数字开头，不能使用JavaScript中的关键字、保留字等
    • 合法的标识符
      • indentifer、username、userName、user_name、$username
    • 非法的标识符
      • int、3.14、hello world
    • JavaScript的标识符大小写敏感
      • Name、name

    2.代码格式

    • 每条功能执行语句的末尾必须加上分号（;）(不一定)
    • 每个单词之间使用空格、制表符、换行符分隔，一般是使用空格分隔

    3.注释

    • 单行注释
      • //
    • 多行注释
      • /* … */

    4.基本数据类型与常量

    • 整型常量
      • 十六进制的格式：0x4f
      • 八进制的格式：0123
      • 十进制的格式：123
    • 实型常量
      • 12.34、3.14、5E7、7e8
    • 布尔值常量
      • true、false
    • 字符串常量
      • “javaScript” 、’hello world’
      • “” 、 ‘’
      • “\n”换行、 “\r”回车、 “\t”制表、 “\\“
      • “hello”+”world”（处理比较长的字符串）
    • null常量
      • 一个变量所指向的对象为空值
      • 可以理解为已经为这个变量赋值，只是这个值是一个空值
    • undefined常量
      • 一个变量还没有被赋值的一个状态，或者表示某对象的某一个属性不存在
    • null常量与undefined常量的区别
      • null确实已经赋值，只是这个值是空的
      • undefined确实没有赋值，或者根本就不存在

    5.变量

    • 定义一个变量，系统就会为之分配一块内存，程序可以用变量名来表示这块内存中的数据
    • 声明变量使用关键字var
      • var name;
        • 此时的name类型就是undefined的
    • 声明变量与赋值一起使用
      • var age = 18;
    • 对已经赋过值的变量再赋其他类型的值（弱类型）
      • var name = “zhangsan”;
      • name = 321;
    • 不事先声明变量，直接使用
      • x = 123;
      • name = “zhangsan”;
    • 下面y的值应该是多少？
      • x = 123;
      • y = x+1;
      • alert(“y=”+y); 124
    • 建议不要采取不声明变量就直接使用的方式，降低程序代码的可读性.

    四、运算符

    1.算术运算符

    +加法运算符(正值运算符)

    x+5、+6

    多字符串相拼接

    “x”+123 -> “x123”

    “”+123 -> “123”

    -加法运算符(负值运算符)

    x-5、-6

    *乘法运算符

    4*6 -> 24

    /除法运算符

    9/4 -> 2.25

    %模运算符

    9/4 - > 1

    ++

    将变量的值加1后再将结果赋值给这个变量

    有前++与后++之分

    a = 2;b = a+++1;c = ++a +1;

    alert(“b=”+b+”,c=”+c);

    –

    将变量的值加减1后再将结果赋值给这个变量

    有前–与后–之分

    a = 2;b = a–+1;c = –a +1;

    alert(“b=”+b+”,c=”+c);

    2.赋值运算符

    赋值运算符的作用就是将一个值赋给一个变量，最常用的赋值运算符是”=”，还可以由”=”赋值运算符与其他一些运算符组合产生一些新的赋值运算符，如”+=”、 “-=”、 “*=”、 “=“等等

    +=

    将变量与所赋的值相加后的结果再赋给该变量

    x+=3，等价于x=x+3;

    3.比较运算符

    >

    当左操作数大于右操作数的时候返回true，否则返回false

    <

    当左操作数小于右操作数的时候返回true，否则返回false

    >=

    当左操作数大于或者等于右操作数的时候返回true，否则返回false

    <=

    当左操作数小于或者等于右操作数的时候返回true，否则返回false

    ==

    当左操作数等于右操作数的时候返回true，否则返回false

    !=

    当左操作数不等于右操作数的时候返回true，否则返回false

    4.逻辑运算符

    &&（逻辑与）

    当左右两边的操作数都为true时返回true，否则返回false

    ||（逻辑或）

    当左右两边的操作数都为false时返回false，否则返回true

    !（逻辑非）

    当操作数为true时返回false，否则返回true

    ()

    可以改变运算符的优先级别

    无需去死记运算符的默认优先级别，即使是记不住，也可以使用括号的方式达到想要的级别

    七、对象

    对象与对象实例

    • 对象中所包含的变量就是对象的属性，对象中所包含的函数就是对象的方法，对象的属性和方法都叫对象的成员.

    • 对象是对某一类事物的描述，是抽象的概念

    • 对象实例是一类事物的具体个例

    • 能够被用来创建对象实例的函数叫做对象的构造方法，只要定义了一个对象的构造函数，就等于定义了一个对象，使用new关键字和对象的构造函数就可以完成一个对象实例的创建

      <script>
    function Student(){}

    var stu = new Student();
    stu.name = "zhangsan";
    stu.age = 23;

    function introduct(){
        alert(stu.name+"今年"+stu["age"]+"岁");
    }

    stu.introduct = introduct;
    stu.introduct();
</script>
    • function Student(){}
      • 定义了一个构造函数，跟普通的函数很类似，为了区别开，我们这里使用了首字母大写的方式
    • var stu = new Student();
      • 调用了Student()这个构造函数创建了一个Student对象的实例，并赋值给了stu变量
    • stu.name = “zhangsan”;
    • stu.age = 25;
      • 动态地为该实例添加了name和age两个属性
      • 这个比Java要更加的随意，居然属性还能动态添加？强大!
    • function introduct(){ alert(stu.name+”今年”+stu[“age”]+”岁”); }
      • 定义了一个普通的函数introduct
      • 对象实例属性的访问方式有两种
        • 对象实例变量.属性名
        • 对象实例变量[“属性名”]，这种方式比较灵活，因为”属性名”可以被一个变量所代替，这样可以以一行代码的方式访问所有的对象实例属性
    • stu.introduct = introduct;
      • 这里又为对象实例动态地添加了introduct方法，并将之前的introduct函数直接赋值给它
    • stu.introduct();
      • 访问对象实例的方法

    构造方法与this关键字

    • 为一个对象实例新增的属性和方法，不会增加到同一个对象所产生的其他对象实例上

      <script>
	function Student(){}
    var stu1 = new Student();
    var stu2 = new Student();
    stu1.name = "zhangsan";
    stu1.age = 23;
    alert(stu1.name);
    alert(stu2.name);
</script>

    • 所有的实例对象在创建后都会自动调用构造函数，在构造函数中增加的属性和方法会被增加到每个对象实例上

    • 对象实例是用new关键字创建的，在构造方法中不要有返回结果的return语句

    • 调用对象的成员方法的时候，需要使用“对象实例.成员方法”的形式，用作成员方法的函数被调用时，一定伴随有某一个对象实例。this关键字代表某个成员方法执行时，引用该方法的当前对象实例，所以，this关键字一般只用在对象成员方法中

      <script>
	function Student(name,age){
        this.name = name;
        this.age = age;
        this.introduct = introduct;
    }
    function introduct(){
        alert(this.name+"今年"+this.age+"岁");
    }
    var stu1 = new Student("zhangsan",25);
    var stu2 = new Student("lisi",24);
    
    stu1.introduct();
    stu2.introduct();
</script>

    在函数中修改参数值

    • 将基本数据类型的变量作为函数参数传递的情况

      <script>
    var x = 3;
    function changeValue(x) {
        x = 5;
    }
    changeValue(10)
    alert(x);//3
</script>

    • 将对象数据类型的变量作为函数参数传递的情况

      <script>
        function Student(name,age) {
            this.name = name;
            this.age = age;
        }
        function changeValue(stu) {
            stu.name = "lisi";
            stu.age = 26;
        }
        var stu = new Student("zhangsan",25);
        changeValue(stu);
        alert(stu.name+","+stu.age);//lisi,26
    </script>

    • 将基本数据类型的变量作为函数参数传递的时候，会将该值拷贝一份传递到函数内部去执行，不管对该拷贝的值做了任何的修改，都不会对原来的值造成任何的修改

    • 将对象数据类型的变量作为函数参数传递的时候，会将该值也拷贝一份传递到函数内部去执行，只是这个拷贝与这个原来的值都是指向同一个对象实例，任何一方对其改变， 都会造成改变

### 数组

#### 数组列表

- 用于表示一组数据的集合
- 由一对方括号（[ ]）包围，列表中的每一个元素用逗号分隔
- 数组元素可以是任意类型的数据，甚至是另外一个数组
  - var arr = ["zhangsan",25,true,'m'];
- 对于每个数组变量，都有一个length属性，表示该数组中元素的个数
- 使用“数组变量名[索引号]”的格式来访问每个数组元素
- 数组列表中的第一个元素的索引号是0
- 数组列表中的最后一个元素的索引号是数组length属性值 -1

#### 用对象的方式实现数组

```html
<script>
	function MyArray(){
        this.length = arguments.length;
        for(var i=0;i<this.length;i++){
            this[i] = arguments[i];
        }
    }
    var str = "";
    var arr = new MyArray(1,2,3,4,"javaScript");
    for(var i=0;i<arr.length;i++){
        str += arr[i]+" ";
    }
    alert(str);
</script>
