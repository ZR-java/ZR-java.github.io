    一、DHTML

    1.DHTML的概念

    • JavaScript将浏览器本身、网页文档、以及网页文档中的HTML元素等都用相应的内置对象来表示，这些对象以及对象之间的层次关系成为DOM（Document Object Model，文档对象模型）

    • 用户通过鼠标或按键在浏览器窗口或网页元素上执行的操作，对DOM对象来说，就称之为事件（Event）。如，用户用鼠标单击了网页上面的某一个按钮，在这个按钮上就发生了鼠标单击事件，按钮就是事件源.

    • 如果将一段程序代码与某一个事件源上发生的某种事件相关联，当该事件源上发生该事件时，浏览器会自动执行与之关联的程序代码，从而引发的一连串程序动作，这个过程称之为事件驱动（Event Driver）。对事件进行处理的程序或者函数，被称之为事件处理程序（Event Handler），它完成对事件进行响应的动作

    • CSS、脚本编程语言和DOM的结合使用，能够使HTML文档与用户具有交互性和动态变换性，这三种技术的统一称谓叫做DHTML（Dynamic HTML，动态HTML）

      观察者模式：

      • 事件源对象：发出事件的对象
      • 事件对象：事件
      • 监听器对象：针对某一个被监听 事件源对象 的具体 事件对象 所做出的处理

    3.编写事件处理程序

    • 方式一

      • 在事件源对象所对应的html标签上增加一个要处理的事件 属性，让事件属性值等于处理该事件的函数名或者是程序代码

        • JavaScript代码

          function fun(){
    alert("哈哈~~");
}

        • 网页代码

          <input type="button" value="点击我" onclick="fun()" />

    • 方式二

      • 直接在html标签属性值中编辑JavaScript代码（现在不需要写JavaScript）

        <input type="button" value="点击我" onclick="javaScript:alert('哈哈~')" />

    • 方式三

      • 直接在JavaScript代码中，设置元素对象的事件属性，让事件属性值等于处理该事件的函数名或者程序代码

        <html>
    <head>
        <script>
            window.onload = function () {
                var btn = document.getElementById("btn");
                btn.onclick = fun;
            }
            function fun(){
                alert("哈哈~~");
            }
   
        </script>
    </head>
    <body>
        <input id="btn" type="button" value="点击我" /><br/>
    </body>
</html>

    • 方式四

      • 在一个专门的<script>标签对中编写某个元素对象的某种事件处理程序代码，并用for属性指定事件源和用event属性指定事件名

        <html>
    <head>
        <script for="window" event="onload">
        	alert("网页加载成功!");
        </script>
    </head>
    <body>
        我就是一个普通的网页
    </body>
</html>

    二、window对象

    • window对象代表浏览器的整个窗口，我们可以利用window对象控制浏览器窗口的各个方面，如改变状态栏上的显示文字、弹出对话框、移动窗口的位置等
    • 对window对象的属性和方法的引用，可以省略”window.”这个前缀，如，window.alert(“你好”)可以直接写成alert(“你好”)

    1.window对象的方法

    • alert ( )
      • 弹出一个有”确定”按钮的对话框
    • confirm ( )
      • 弹出一个有”确定”和”取消”两个按钮的对话框，点击”确定”按钮会返回true，点击”取消”按钮会返回false，编程人员可以根据具体的返回值进行进一步业务逻辑的处理
    • prompt ( )
      • 弹出一个具有一个输入框和一个”确定”按钮的对话框，并返回用户所输入的内容
    • setInterval ( )
      • 设置每隔多少时间（毫秒）调用一次某一个程序代码
    • setTimeout ( )
      • 设置多长时间（毫秒）以后执行某一段程序代码
    • clearInterval ( )
      • 取消setInterval的设置效果，其参数是setInterval方法的 返回值
    • clearTimeout ( )
      • 取消setTimeout的设置效果，其参数是setTimeout方法 的返回值
    • moveTo ( )
      • 将浏览器窗口移动到屏幕上的某一个位置
    • resizeTo ( )
      • 改变浏览器窗口的大小
    • open ( )
      • 打开一个新的窗口
    • showModalDialog ( )
      • 打开一个模态的一个对话框窗口，必须关闭该窗口才能回到原来的浏览器窗口
    • showModelessDialog ( )
      • 打开一个非模态的一个对话框窗口，无需关闭该窗口就能回到原来的浏览器窗口

    实例举例

    • 打开一个网页的同时再打开一个新的窗口，该窗口是发布一个通知，然后在一定的时间之后，该窗口自动关闭

      <html>
    <head>
        <script>
        	window.open("msg.html","_blank","top=0,left=0,width=200,height=200,toolbar=no");
        </script>
    </head>
    <body>
        我就是一个普通的网页
    </body>
</html>

    • 打开一个网页的同时再打开一个新的窗口，该窗口是发布一个通知，然后在一定的时间之后，该窗口自动关闭

      <html>
    <head>
        <script>
        	window.setTimeout("window.close()",5000);
        </script>
    </head>
    <body>
        这是通知页面<br/>
        大家好,JavaScript的DOM编程是不是很有意思?<br/>
        (5秒后关闭该页面)
    </body>
</html>

    2.window对象的属性

    • closed
      • 窗口是否关闭
    • opener
      • 打开当前窗口的window对象
    • defaultstatus
      • 默认设置的内容
    • status
      • 窗口状态栏中正在显示的内容
    • screenTop
      • 窗口顶点在垂直方向上的位置
    • screenLeft
      • 窗口顶点在水平方向上的位置
    • location
      • 用于窗口或框架的 Location 对象 (url)
    • history
      • 对 History 对象的只读引用
        • back ( ) : 加载history列表中的前一个URL
        • forward ( ) : 加载history列表中的下一个URL
        • go ( ) : 加载history列表中的某一个具体页面

    3.window对象的事件

    • onload事件
      • 浏览器装载网页文档完毕之后（用户还未看到网页文本之 前）
    • onunload事件
      • 浏览器卸载网页文档之后
    • onbeforeunload事件
      • 浏览器准备卸载网页文档之时
    <html>
    <head>
        <script>
        	window.onload = function(){
                alert("网页文档装载完毕");
            }
        </script>
    </head>
    <body>
        我就是一个普通的网页
    </body>
</html>

    三、document对象

    document对象的方法

    • wirte ( )
      • 向html文档中动态写入内容
    • writeln ( )
      • 向html文档中动态写入内容，末尾多增加一个换行符，然而并不能看到效果
    • getElementById ( )
      • 根据元素的id值返回这个元素对象
    • getElementsByName ( )
      • 根据元素的name值返回这个元素对象，这个返回值是一个数组
    • getElementsByTagName ( )
      • 根据元素的标签名返回这个元素对象，这个返回值是一个数组
    • createElement ( )
      • 创建一个HTML元素对象
    • createStyleSheet ( )
      • 创建一个CSS样式规则
    • open ( )
      • 打开一个新的文档，与window的open类似，建议使用 window的open方法
    • close ( )
      • 关闭一个文档
    • clear ( )
      • 清除文档中的所有内容（不成熟）

    1.body对象的方法

    • appendChild ( )

      • 将一个标签对增加到文档末尾处

        <html>
    <head>
        <script>
        	function createA(){
                var a = document.createElement("A");
                a.href = "http://www.baidu.com";
                a.innerText = "百度";

    <body onload="createA()">
        我就是一个普通的网页

    五、form对象

    form对象最主要的功能就是能够直接访问HTML文档中的form表单。

    一个web页面可以有一个或者多个form表单，使用document.forms数组对象可以访问到各个form表单对象。

    可以将标签中嵌入的表单字段元素的名称（name属性的值）作为form对象的属性，来引用该表单字段元素所对应的对象

    1.form对象的方法

    • submit ( )
      • 该方法是form对象的专有方法，用于向www服务器递交表单数据，模拟用户点击<input type="submit".../>按钮的效果
      • 该方法不会产生onsubmit事件
    • item ( ) HTML5已弃用
      • 返回代表form表单中的某个表单字段元素所对应的对象，接收的参数可以是表单字段元素的名称，也可以是表单字段元素在form表单中的索引号.
      • item方法不能返回<input type="image".../>类型的表单字段元素

    2.form对象的属性

    • name
    • target
    • title
    • enctype
    • encoding
    • method
    • action
    <form name="form1" action="http://www.baidu.com" method="get">
    姓名:<input type="text" id="name" name="name" /><br/>
    密码:<input type="password" id="password" name="password" /><br/>
    年龄:<input type="text" id="age" name="age" /><br/>
    性别:<input type="text" id="sex" name="sex" /><br/>
    学历:<input type="text" id="degree" name="degree" /><br/>
</form>
<a href="javascript:submitForm()">提交</a>
    <script language="javascript">
    function submitForm(){
        form1.method = "post";
        form1.action = "";
        var name = document.form1.name.value;//可读性太低
        var password = document.form1.item("password").value;
        var age = document.forms[0].item(2).value;
        var sex = document.getElementById("sex").value;//用这个,以上都不行
        var degree = document.getElementByName("degree")[0].value;
        alert("name="+name+"\npassword="+password+"\nage="+age+"\nsex="+sex+"\ndegree="+degree);
        document.forms["form1"].submit();
    }
</script>

    3.form对象的事件

    • onsubmit

      • 表单验证

      • 调用格式

        <form ... onsubmit="return validate(this)">

        function validate(form){
	var nameTag = form.name;
    var passwordTag = form.password;
    if(nameTag.value==""){
        alert("姓名不能为空");
        nameTag.focus();
        return false;
    }
    if(passwordTag.value==""){
        alert("姓名不能为空");
        passwordTag.focus();
        return false;
    }
    return true;
}
