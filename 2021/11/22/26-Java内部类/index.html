Java内部类

内部类是定义在另外一个类中的类

  • 内部类方法可以访问所在外部类的属性，包括私有属性
  • 内部类可以对同一个包中其他类隐藏
  • 可以使用匿名内部类减少代码量
  • 主要的用处是在处理事件监听，其他地方建议避免使用内部类
  • 理解内部类的方式类似于理解变量

静态内部类

最简单的一种内部类

  • 名称不能跟外部类的名称一样
  • 编译的时候会跟外部类分开编译，外部类一个class文件， 内部类一个class文件
  • 能访问外部类的静态的属性和方法，包括private的
  • 创建静态内部类对象的格式
    • new outerClass.innerClass();
    • 如果在外部类的范围之内访问该类的内部类,可以不需要前缀outerclass

InnerClassStatic.java

/*
静态内部类可以访问外部类的
	静态变量
	静态方法
静态内部类可以访问自己类的
	静态or普通成员变量
	静态or普通方法
*/
public class InnerClassStatic{
	
	public static int a1 = 10;
	private int a2 = 20;
	
	public static void test1(){
		System.out.println("InnerClassStatic:static:test1()");
	}
	
	public void test2(){
		System.out.println("InnerClassStatic:member:test2()");
	}
	//静态内部类
	public static class InnerClass{
		
		public static int a1 = 30;
		private int a2 = 40;
		
		public static void test1(){
			System.out.println("InnerClass:static:test1()");
		}
		
		public void test2(){
			System.out.println("InnerClass:member:test2()");
		}
		
		public void test(){
			
			System.out.println("InnerClassStatic.a1="+InnerClassStatic.a1);
			InnerClassStatic.test1();
			/*
			错误,静态内部类无法从静态上下文中引用非静态变量,方法
			System.out.println("InnerClassStatic.a2="+InnerClassStatic.this.a2);
			InnerClassStatic.test2();
			*/
			System.out.println("======================");
			
			System.out.println(a1);
			System.out.println(a2);
			test1();
			test2();
		}
	}
	
	public static void main(String[] args){
		//静态调用用类就行
		InnerClassStatic.InnerClass instance = new InnerClassStatic.InnerClass();
		instance.test();
	}
	
}

成员(非静态)内部类

跟静态内部类定义一样,仅仅去掉static关键系

  • 可以访问外部类的所有的属性和方法（静态、非静态）
  • 在外部类中访问成员内部类的方式
    • this.new Innerclass();
  • 在其他类中访问成员内部类的方式
    • (new Outerclass()).new Innerclass();
  • 访问外部类属性、方法的方式
    • Outerclass.this.member

InnerClassMember.java

/*
成员内部类能访问外部类的
	静态变量
	静态方法
	普通成员变量
	普通方法
	
成员内部类不能有静态属性和方法
*/

public class InnerClassMember{
	
	public static int a1 = 10;
	private int a2 = 20;
	
	public static void test1(){
		System.out.println("InnerClassMember:static:test1()");
	}
	
	public void test2(){
		System.out.println("InnerClassMember:member:test2()");
	}
	
	//成员内部类
	public class InnerClass{
		
		/*
		不能有静态成员变量和方法
		public static int a1 = 30;
		public static void test1(){
			System.out.println("InnerClass:static:test1()");
		}
		*/
		private int a2 = 40;
		
		public void test2(){
			System.out.println("InnerClass:member:test2()");
		}
		
		public void test(){
			System.out.println("InnerClassMember.a1="+InnerClassMember.a1);
			System.out.println("InnerClassMember.a2="+InnerClassMember.this.a2);
			InnerClassMember.test1();
			InnerClassMember.this.test2();
			
			System.out.println(a2);
			test2();
		}
	}
	
	public static void main(String[] args){
		//成员内部类调用需要对象
		InnerClassMember.InnerClass instance = new InnerClassMember(). new InnerClass();
		instance.test();
	}
	
}

局部内部类

  • 定义在方法中
  • 用得相对较少，但是面试问得最多
  • 前面不能有修饰符修饰（private、protected、public、 static等）
  • 能访问外部类所有的属性和方法（包括静态的），以及final局部变量

InnerClassPart.java

/*
局部内部类不能有静态属性和方法
局部内部类能访问外部类的
	静态变量
	静态方法
	普通成员变量
	普通方法
*/
public class InnerClassPart{
	
	public static int a1 = 10;
	private int a2 = 20;
	
	public static void test1(){
		System.out.println("InnerClassPart:static:test1()");
	}
	
	public void test2(){
		System.out.println("InnerClassPart:member:test2()");
	}
	

	public void test(){
		//局部内部类
		class InnerClass{
			
			//public static int a1 = 30;	不能有静态属性
			
			private int a2 = 40;
			
			/*
			不能有静态方法
			public static void test1(){
				System.out.println("InnerClass:static:test1()");
			}
			*/
			
			public void test2(){
				System.out.println("InnerClass:member:test2()");
			}
			
			public void test(){
				System.out.println("InnerClassPart.a1="+InnerClassPart.a1);		
				System.out.println("InnerClassPart.this.a2="+InnerClassPart.this.a2);
				InnerClassPart.test1();
				InnerClassPart.this.test2();
			
				System.out.println(a2);
				test2();
			}
		}
		
		InnerClass instance = new InnerClass();
		instance.test();
		
	}
	
	
	public static void main(String[] args){
		
		InnerClassPart instance = new InnerClassPart();
		instance.test();
	}
	
}

匿名内部类

  • 没有名字的局部内部类 (说明定义在方法内部)
  • 没有class关键字
  • 没有extends、implements关键字
  • 没有构造器
  • 隐式的继承一个父类或者实现一个接口

new了一个匿名内部类，就是new了一个 实现了接口 的 实现类 的对象

OneInterface.java

public interface OneInterface{
	String getVersion();
}

InnerClassNoName.java

public class InnerClassNoName{
	
	public static void main(String[] args){
		
		OneInterface oi1 = new OneInterface(){ //匿名内部类的规范写法(new的是接口)
			@Override
			public String getVersion(){
				return "2.0";
			}
		};
		System.out.println(oi1.getVersion());
		
		Runnable runnable = new Runnable(){
			@Override
			public void run(){
				System.out.println("run...");
			}
		};
		runnable.run();
		
	}
	
}
