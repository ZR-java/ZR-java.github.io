    流的概念

    流的分类 (三个要素)

    • 处理方式 (字节和字符)
      • 字节可以操作任何格式的文件
      • 字符只能操作字符格式的文件
    • 流的方向 - 参照物:应用程序
      • InputStream : 可以通过该流将外部的数据读进我们的程序中
      • OutputStream : 可以通过该流将我们的程序中的数据输出到外部
    • 功能
      • 如果这个流是处理其他流的，就是处理流（过滤流）
      • 如果这个流不处理其他流，就不是处理流（过滤流），是节点流
      • 节点流，就是直接“搭”在数据上面的，节点流操作数据的来源，而处理流操作节点流。

    流的分类区分方式

    区分输入流

    • 输入流
      • 以InputStream、Reader结尾
    • 输出流
      • 以OutputStream、Writer结尾

    区分字节流、字符流

    • 字节流
      • 以Stream结尾
    • 字符流
      • 以Reader结尾

    区分节点流、处理流

    • 看构造器里面的参数
    • 节点流
      • 构造器参数是String、File等
    • 处理流
      • 构造器参数有其他流

    InputStream的方法、OutputStream的方法

    InputStream

    • int read()

    • int read(byte[])

    • int read(byte[], int, int)

    • void close()

    • void mark(int)

    OutputStream

    • void write(int)
    • void write(byte[])
    • void write(byte[], int, int)
    • void close()
    • void flush()

    Reader、Writer的方法

    Reader

    • int read()
    • int read(char[])
    • int read(char[], int, int)

    Writer

    • void write(int)
    • void write(char[])
    • void write(char[], int, int)
    • void write(String)
    • void write(String, int, int)
    • void close()
    • void flush()

    键盘输入

    BufferedReader、BufferedWriter

    三部曲:带缓冲的字符输入/输出流

    BufferedReader br = new BufferedReader(//缓冲字符输入流
        new InputStreamReader(//字节流转换字符流,过渡流
            new FileInputStream(inputpath)//文件输入字节流
        )
);
    public static void main(String[] args) throws Exception{

        BufferedReader keyInput = new BufferedReader(
                new InputStreamReader(
                        System.in	//标准键盘输入流
                )
        );
        while (true){
            System.out.println("请输入一段文字(exit:退出)");
            String inputcontent = keyInput.readLine();	//读一行
            if (inputcontent.equals("exit")){
                System.out.println("再见");
                System.exit(0);
            }else {
                System.out.println("你输入的内容是:"+inputcontent);
            }
        }

}

    编码格式

    ASCII（一个英文字母（不分大小写）占一个字节，一个中文汉字占两个字节）

    • ISO8859-1
    • ISO8859-2
    • ISO8859-3

    Unicode（一个英文等于两个字节，一个中文（含繁体）等于两个字节）

    • GBK（英文占1个字节，中文占两个字节）
    • GB2312
    • Big5
    • UTF-8（一个英文字符等于一个字节，一个中文（含繁体）等于三个字节）
