数组的基本操作

常见的数据结构

  • 数组(Array):管理多个数据
  • 栈(Stack):先进后出(后进先出)
  • 队列(Queue):先进先出
  • 链表(Linked List):查询效率低，增加、删除效率高
    • 单向链表
    • 双向链表
    • 环形链表
  • 二叉数等………..

数组概述

  • 数组也是对象
  • 数组中保存着多个相同类型的元素
  • 数组中的每一个元素都是变量

数组类型的默认值

  • byte、short、int、long为0
  • float、double为0.0
  • char为’\u0000’
  • boolean为false
  • 引用类型为null

Java语言注重类型

  • 一旦数组被声明出来,你就只能够装入所声明类型的元素
  • 数组元素的初始化分为两种
    • 基本类型元素的初始化就是直接将数据放入数组元素中
    • 引用类型的元素的初始化需要去指向另外一个对象

数组的拷贝(扩容)

System.arraycopy(); (java.lang下的)

arraycopy(Object src, int srcPos, Object dest, int destPos, int length)

  • src - 源数组
  • srcPos - 源数组中的起始位置
  • dest - 目标数组
  • destPos - 目标数据中的起始位置
  • length - 要复制的数组元素的数量

数组排序

冒泡排序 (效率最低):

  1. 比较相邻的元素。如果第一个比第二个小，就交换他们两个。
  2. 对每一对相邻元素作同样的工作，从开始第一对到结尾的最后一对。在这一点，最后的元素应该会是最小的数。
  3. 针对所有的元素重复以上的步骤，除了最后一个。
  4. 持续每次对越来越少的元素重复上面的步骤，直到没有任何一对数字需要比较。

创建TestBubbleSort

//冒泡排序
import java.util.Random;   //随机数

public class TestBubbleSort{
	
	public static void main(String[] args){
		
		Random random = new Random();
		int[] array = new int[10];
		//使用随机数为数组元素赋值
		for(int i=0;i<array.length;i++){
			array[i] = random.nextInt(100);
		}
		//遍历原始数组
		System.out.print("遍历原始数组:");
		for(int n:array){
			System.out.print(n+" ");
		}
		//排序
		int temp;
		for(int i=0;i<array.length;i++){
			for(int j=0;j<array.length-1-i;j++){	//-i步骤优化了程序,
													//冒泡排序一趟下来最后两位不要动了
				if(array[j] > array[j+1]){
					temp = array[j];
					array[j] = array[j+1];
					array[j+1] = temp;
				}
			}
		}
		//遍历排序后的数组
		System.out.print("\r\遍历排序后数组:");
		for(int n:array){
			System.out.print(n+" ");
		}
		
	}	
	
}

插入排序:

  1. 从第一个元素开始，该元素可以认为已经被排序
  2. 取出下一个元素，在已经排序的元素序列中从后向前扫描
  3. 如果该元素（已排序）大于新元素，将该元素移到下一位置
  4. 重复步骤3，直到找到已排序的元素小于或者等于新元素的位置
  5. 将新元素插入到该位置中
  6. 重复步骤2 
import java.util.Random;

public class TestInsertSort{
    
    public static void main(String[] args){
        
		Random random = new Random();
		int[] array = new int[10];
		//使用随机数为数组元素赋值
		for(int i=0;i<array.length;i++){
			array[i] = random.nextInt(100);
		}
		//遍历原始数组
		System.out.print("遍历原始数组:");
		for(int n:array){
			System.out.print(n+" ");
		}
		//排序
		int size = array.length;
		int temp;
		for(int i=1;i<size;i++){
			temp = array[i];
			int j=i;
			//找到位置往后挪
			while(j>0){
				if(temp < array[j-1]){
					array[j] = array[j-1];
				}else{
					break;
				}
				j--;
			}
			//插入数据
			array[j]=temp;
		}
		
		//遍历排序后的数组
		System.out.print("\r\n遍历排序后数组:");
		for(int n:array){
			System.out.print(n+" ");
		}
		
    }
    
}

选择排序:

  1. 在未排序序列中找到最小的元素,存放到排序序列的起始位置
  2. 再从剩余未排序元素中继续寻找最小元素,然后放到排序序列起始位置
  3. 以此类推,直到所有元素均排序完毕

创建TestSelectSort

import java.util.Random;

public class TestSelectSort{
    
    public static void main(String[] args){
        
		Random random = new Random();
		int[] array = new int[10];
		//使用随机数为数组元素赋值
		for(int i=0;i<array.length;i++){
			array[i] = random.nextInt(100);
		}
		//遍历原始数组
		System.out.print("遍历原始数组:");
		for(int n:array){
			System.out.print(n+" ");
		}
		
		//排序
		int temp;
		int size = array.length;
		for(int i=0;i<size;i++){
			int k=i;
			for(int j=size-1;j>i;j--){
				if(array[j] < array[k]){
					k = j;
				}
			}
			temp = array[i];
			array[i] = array[k];
			array[k] = temp;
		}
		
		//遍历排序后的数组
		System.out.print("\r\n遍历排序后数组:");
		for(int n:array){
			System.out.print(n+" ");
		}
		
    }
    
}

快速排序:

  1. 从数列中挑出一个元素，称为“基准”(pivot)
  2. 重新排序数列，所有元素比基准值小的摆放在基准前面，所有元素比基准值大的摆在基准的后面（相同的数可以到任一边）。在这 个分割之后，
  3. 该基准是它的最后位置。这个称为分割（partition）操作。
  4. 递归地把小于基准值元素的子数列和大于基准值元素的子数列排序。
public class TestQuickSort{
	public static void QuickSort(int arr[],int L,int R)
	{
		if(L>=R)
			return;
		int left=L,right=R;
		int pivot = arr{left];
		while(left<right){
			while{left<right && arr[right]>=pivot){
				right--;
			}
			if(left<right){
				arr[left] = arr[right];
			}
			while(left<right && arr[left]<=pivot){
				left++;
			}
			if(left<right){
				arr[right] = arr[left];
			}
			if(left>=right){
				arr[left] = pivot;
			}
		}
		QuickSort(arr,L,right-1);
		QuickSort(arr,right+1,R);
	}
}
