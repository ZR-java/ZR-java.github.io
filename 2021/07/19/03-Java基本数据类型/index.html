    • 03-Java基本数据类型

    2021-07-19- 2022-04-07
    1.3k- 5m
    JavaSE-Java

    Java基本数据类型

    byte 8bit整型值

    short 16bit整型值

    int 32bit整型值

    long 64bit整型值

    float 32bit浮点数值

    double 64bit浮点数值

    char 16bit unicode编码

    boolean 布尔值(true或false)

    1、整型数据类型：byte、short、int、long

    • 三种形式给一个整型变量直接赋值：十进制、 八进制、十六进制
    • 直接给出的整型值默认是int类型
    • 长整型值应该在数字后面跟上 ‘l’ 或 ‘L’ 标识

    整型数据在计算机内的存储方式

    • 计算机以补码的形式保存所有的整数
    • 补码计算规则:
      • 正数的补码和原码完全相同
      • 负数的补码是其反码加1
      • 反码是对其原码按位取反,符号位保持不变

    2、浮点数据类型:float和double

    • 直接在程序中书写的浮点数默认认为double类型
    • float类型的数据占32位长度(4 Bytes)
    • double类型的数据占64位长度(8 Bytes)
    • 直接表示一个float类型的值必须在数字后跟 ‘f’ 或 ‘F’
    • 直接表示一个double类型的值可以在数字后跟 ‘d’ 或 ‘D’
    • 可以用科学计数法表示,如:123.456e3 或 123.456E3

    3、字符类型:char

    • 其实就是一个无符号的16位整型,其范围为0~65535,用来存储该字符的Unicode编码值,因为Unicode编码中包含汉字的编码,所以一个char类型的变量也可以存储一个汉字字符.(一个汉字字符=两个字节)

    • 在代码中直接给出的一个字符值必须用单引号括起来

      char zhChar = ‘中’;

      char enChar = ‘a’;

      char ucChar = ‘\u0060’;

    • 对某些不方便直接书写或直接书写有歧义的字符,Java语言规定用转义字符给出

    • 可以把char类型数据当作整数数据来操作

    转义字符

    • Java语言中转义字符都以 ‘\‘ 给出

    • 常用转义字符

      \uxxxx :1到4位16进制数所表示的字符(xxxx)

      \‘ :单引号字符

      \“ :双引号字符

      \\ :反斜杠字符

      \r :回车

      \n :换行

      \t :制表符

    转义字符 描述
    \ddd 1到3位8进制数所表示的字符(ddd)
    \uxxxx 1到4位16进制数所表示的字符(xxxx)
    \‘ 单引号字符
    \’’ 双引号字符
    \\ 反斜杠字符
    \r 回车
    \n 换行
    \f 走纸换页
    \t 横向跳格
    \b 退格

    char类型数据当作整数数据

    char c1 = '中',c2 = 54,c3 = 'A',c4 = 'Z';
char c5 = 'a',c6 = 'z',c7 = '0',c8 = '9';
int i1 = 'A';i2 = 'a',i3 = '0';
System.out.println((int)c3);
System.out.println(i1);
System.out.println(i2);
System.out.println(i3);

    4、布尔类型:boolean

    • 一个boolean类型的变量用来表示一个或真或假的关系
    • 一个boolean类型的变量的值必须是true或false二者之一
    boolean isCorrect = true;
    类型 位数 字节 最小值 最大值
    boolean 8 1 true或false true或false
    byte 8 1 -27 27-1
    short 16 2 -215 215-1
    char 16 2 0 216-1
    int 32 4 -231 231-1
    long 64 8 -263 263-1
    float 32 4 Not needed Not needed
    double 64 8 Not needed Not needed

    6、自动类型转换

    基本数据类型转换序列

    • 顺着箭头方向的类型转换可以自动进行,逆箭头方向的转换或无箭头类型之间的转换必须强制转换
    • 能将A类型变量自动转换成B类型变量的前提条件是这种转换不会造成数据和精度的丢失,即A类型的取值范围是B类型的取值范围的子集

    例子:

    public void implictCasting(){
    byte a = 60;
    int ia = a;
    char b = 'a';
    int c = b;
    long d = c;
    long e = 100000000L;
    float f = e;
    double g = f;
    String s = "Hello";
}

    自动类型转换的规则

    • 所有的byte类型、short类型、char类型被升级到int类型

    • 在算术过程中，会自动提升到最高等级的类型

      byte b1 = 12;
byte b2 = b1+1;  //错误,自动升级到int类型
long value1 = b1+12;
long value2 = b1+12+120L;  //最高级是long类型
int value3 = b1+12+120L;  //错误

    强制类型转换

    • 使用场景
      • 高类型往低类型转换
      • 父类型往子类型转换
      • 强制类型转换操作符:()
    • 注意事项
      • 基本类型的强制类型转换,可能丢失精度
      • 引用类型的强制类型转换,可能转换失败

    7、变量

    变量的类型

    基本类型的变量

    • int age = 25;
    • char c = ‘A’;
    • boolean gender = true;
    • double d = 3.14;

    引用类型的变量

    • String name = “张三丰”;
    • String degree = “本科”;

    变量的声明

    变量的声明,也可以称为变量的定义

    • int age;
    • boolean degree;
    • String name;

    声明变量的规则

    • 变量必须有一个类型
    • 变量必须有一个名字
    • int(类型) age(名字);
